# Чек-лист тестирования калькулятора (на телефоне)

## Базовые арифметические операции
- [ ] Сложение: 2 + 2 = 4
- [ ] Вычитание: 5 - 3 = 2
- [ ] Умножение: 4 × 3 = 12
- [ ] Деление: 10 ÷ 2 = 5

## Граничные случаи
- [ ] Деление на ноль: 10 ÷ 0 → сообщение об ошибке (не зависание)
- [ ] Деление отрицательного числа: (-10) ÷ 2 = -5
- [ ] Умножение на ноль: 5 × 0 = 0
- [ ] Сложение с отрицательным числом: 5 + (-3) = 2

## Работа с большими числами
- [ ] Ввод 8-значного числа (не должно обрезаться)
- [ ] Результат умножения 99999 × 99999 (выводится корректно)

## Последовательные действия
- [ ] 5 + 3 = 8, затем ÷ 2 = 4 (цепочка операций)
- [ ] Кнопка сброса (C или AC) очищает поле

## UI и интерактивность
- [ ] Кнопки реагируют на нажатие (визуальный фидбек)
- [ ] Поворот экрана — интерфейс не ломается
