# [Калькулятор] Ошибка при делении на ноль

**Серьезность:** Critical (критическая)
**Приоритет:** High (высокий)
**Окружение:** Калькулятор на телефоне (Android/iOS — уточнить)

## Шаги для воспроизведения
1. Открыть приложение «Калькулятор».
2. Ввести число `10`.
3. Нажать кнопку деления `÷`.
4. Ввести число `0`.
5. Нажать кнопку равно `=`.

## Ожидаемый результат
Приложение показывает сообщение: «На ноль делить нельзя» или «Error», либо поле результата отображает «Ошибка». Приложение остаётся рабочим.

## Фактический результат
Приложение зависает (не реагирует на нажатия кнопок). Требуется принудительное закрытие и повторный запуск.

## Воспроизводимость
Всегда / 100%

## Дополнительная информация
- Та же ошибка возникает при делении любого числа на ноль: `5 / 0`, `-3 / 0`.
- Деление 0 ÷ 0 также приводит к зависанию.
